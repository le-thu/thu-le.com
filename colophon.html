Typography

System fonts (until I can pick a font worth paying for in the long run) and Newsreader by Production Type.

Tech stack

This site is designed and handcoded by me in good old HTML, CSS, and a little bit of JavaScript. Since the site is pretty simple, I usually prototype directly with code in Sublime Text. I use Figma to test typography and edit images.

Everything is stored on GitHub (repo) and deployed via Cloudflare Pages, with my domain registered on Cloudflare.

Additionally, I collect basic stats using Clicky just to better understand how people find and visit the site.

Photography

All images were taken or created by me, unless stated otherwise.
