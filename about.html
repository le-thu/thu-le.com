Colophon

This site is handcoded by me in good old HTML, CSS, and a little bit of JavaScript. Since the site is pretty simple, I usually design and prototype directly with code in Brackets, and sometimes I use Figma to test typeface options. Typography is set in system fonts, and Newsreader by Production Type. Everything is stored on GitHub and deployed via Cloudflare Pages, with my domain registered on Cloudflare Registrar. Additionally, I collect basic stats using Clicky just to better understand how people find and visit the site.
