Colophon

This site is handcoded by me in good old HTML and CSS, and a little bit of CSS with the help of ChatGPT and Claude. Since the site is pretty simple, I usually just design and prototype directly with code in Brackets, and sometimes I use Figma to test typeface options. Typography is set in system fonts, and Newsreader by Production Type. Everything is stored on GitHub and deployed via Cloudflare Pages. Additionally, I collect basic stats using Clicky just to better understand how people visit the site.
